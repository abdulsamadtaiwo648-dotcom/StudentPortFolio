## 🎯 Project Overview
This website is designed as a personal platform for **Abdulsamad Taiwo**, a Software Engineering student at MIVA Open University. It showcases a professional profile, highlights technical skills, displays real-world projects, and includes an interactive academic task management system.

The project demonstrates proficiency in **HTML, CSS, and JavaScript** while adhering to modern web design principles (responsive layouts, flexbox, grid, interactive UI elements).

---

## 📑 Page Structure

### 1. 🏠 Homepage (`index.html`)
- Professional hero section with an animated typewriter effect and dynamic counters.
- Profile picture with CSS-animated glowing gradients.
- Quick preview of technical skills, featured projects, and an AI-generated voice introduction.

### 2. 👤 About Me (`about.html`)
- Comprehensive overview of career aspirations in Backend Engineering.
- Dynamic, animated skill progress bars.
- Structured educational background table.
- Hobbies and personal interests grid.

### 3. 💼 Projects (`projects.html`)
- A detailed showcase of 3 real-world projects (DLNZ Store, CLI Tower Management, ASCII Art Web).
- JavaScript-powered category filtering system (All, Web, CLI/Systems, Tools).
- Clean card-based CSS Grid layout with hover animations.

### 4. 📋 Academic Planner (`planner.html`)
- Fully interactive task management system built entirely with Vanilla JavaScript.
- Features: Add tasks, set due dates and priorities, mark as complete, or delete.
- Real-time statistics (Total, Pending, Completed) and completion progress bar.
- Data persists across page reloads using browser `localStorage`.

### 5. ✉️ Contact (`contact.html`)
- Professional contact form for networking and inquiries.
- Comprehensive client-side JavaScript validation (empty checks, valid email formats, digits-only phone numbers).
- Simulated form submission with a success state and real-time visual feedback.

---

## 🛠️ Technologies & Features Demonstrated

### HTML
- Extensive use of semantic HTML5 tags (`<nav>`, `<section>`, `<article>`, `<main>`, `<footer>`).
- Well-structured forms, tables, lists, and multimedia (`<audio>`) integrations.

### CSS
- **100% Vanilla CSS** (`style.css`) with CSS custom properties (variables) for consistent theming.
- Modern layout techniques combining **CSS Flexbox** and **CSS Grid**.
- Fully **Responsive Design** utilizing `@media` queries for mobile, tablet, and desktop views.
- Engaging micro-interactions: `@keyframes` animations, hover transitions, and scroll-triggered fade-ins.

### JavaScript
- **DOM Manipulation:** Dynamically rendering tasks and skill bars.
- **Event Handling:** Scroll listeners, form submissions, and button clicks.
- **Form Validation:** Regex-based validation for emails and phone numbers.
- **Data Persistence:** Storing arrays and objects in `localStorage`.
- **Logic:** Array methods (`filter`, `map`, `forEach`) for the project filter and planner stats.

## 💻 Local Installation & Setup

Since this project uses entirely frontend technologies (HTML/CSS/JS) with no build tools required, running it locally is incredibly simple:

1. Clone the repository:
   ```bash
   git clone https://github.com/abdulsamadtaiwo648-dotcom/StudentPortFolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd StudentPortFolio
   ```
3. Open `index.html` in any modern web browser.
   *(For the best development experience, you can use VS Code's "Live Server" extension).*
